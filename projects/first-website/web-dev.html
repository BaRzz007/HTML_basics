What is HTML?

HTML (HyperText Markup Language) is the standard language for defining web pages, which serve as the interface to your web applications. HTML files are plain-text files following the HTML syntax. A browser knows how to take an HTML file and display in a browser window the page defined by the html file. The file can be stored locally on the computer the browser is running on or fetched from a web server usinfg a URL. Creating HTML is coding because the result source code get executed by a browser, which is a virtual machine.

Tags Define the Page Structure

An HTML file consists of content marked-up by tags as shown in this picture

The tag at the beginning of the file is special as it is used to indicate the standard the file is following <!DOCTYPE html> means HTML5 - the latest standard, which some new features. You can copy and paste this tag to every HTML file you create.

Most tags appear in pairs - an opening and a closing tag. Each pair of tags defines an element for the web page, which can be visible. The visible elements are like boxes. All boxes hold some content and some boxes can contain other boxes. For instance, the <html></html> defines the root (hightest level) element, whcih should enclose everything else in your HTML file. Inside the "html" elements are like boxes. All boxes hold some content and some boxes can contain other boxes. For file. Inside the "html" element we have a "head" element and a "body" element defined by their corresponding tags. Inside "head" element is a "title" element and inside the "body" element is a "paragraph" element defined by the <p></p> tag pair.
